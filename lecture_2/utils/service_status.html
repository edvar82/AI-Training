API Gateway Service Disruption

Investigating

September 15, 2023 - 10:23 AM UTC

We are currently investigating issues with our API Gateway service. Users are reporting 503 errors when making API calls. Our engineering team is actively working to identify the root cause. Some regions appear to be more affected than others, with the US-East region showing the highest error rates. We recommend temporarily routing requests through our backup endpoints as documented in our disaster recovery guide.

Database Service Performance Degradation

Issue Identified

September 15, 2023 - 09:15 AM UTC

We have identified the cause of the database service performance degradation. A scheduled maintenance job is causing higher than normal CPU usage on our database clusters. This is resulting in increased latency for database operations, particularly for complex queries. We are working to optimize the maintenance job to reduce its impact. In the meantime, we recommend implementing query caching where possible and avoiding non-essential complex queries until the issue is resolved. We expect to have this issue mitigated within the next 2 hours.

Recent Incidents

Content Delivery Network Outage

Resolved

September 14, 2023 - 14:45 PM UTC

The CDN outage that affected content delivery in the APAC region has been fully resolved. The issue was caused by a configuration error during a routine update to our edge servers. All services have been restored to normal operation, and we have implemented additional validation checks to prevent similar issues in the future. We apologize for any inconvenience this may have caused.

Authentication Service Intermittent Failures

Resolved

September 12, 2023 - 08:30 AM UTC

The intermittent failures in our authentication service have been resolved. The issue was caused by a memory leak in one of our authentication microservices, which has been identified and patched. We have deployed the fix across all regions, and service has returned to normal. We have also added additional monitoring to detect similar issues earlier in the future.

Critical Security Vulnerability Detected

Urgent

September 16, 2023 - 07:30 AM UTC

Our security team has identified a critical vulnerability affecting the authentication service. This zero-day vulnerability could potentially allow unauthorized access to user accounts. We are actively investigating the issue and deploying emergency patches. As a precautionary measure, we recommend enabling two-factor authentication for all admin accounts immediately. We will provide updates as more information becomes available.
